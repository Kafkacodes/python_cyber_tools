# Python Cyber Tools 🔐

Beginner-friendly Python tools for cybersecurity, digital forensics, and automation.

This repository contains small cybersecurity projects built while learning Python and security concepts.

## Tools Included

### 1. Password Strength Checker

Checks password strength based on:

* Length
* Uppercase letters
* Lowercase letters
* Numbers
* Special characters
* Common passwords

### 2. Random Password Generator

Generates secure random passwords with customizable length and character sets.

Features:

* Random password generation
* Letters, numbers, and symbols
* Configurable password length
* Strong password creation

### 3. File Metadata Viewer

Displays file information such as:

* File name
* File size
* Creation time
* Modification time
* File extension
* File or directory detection

### 4. Log Analyzer

Analyzes log files and extracts useful information such as:

* Total log entries
* Failed logins
* Successful logins
* Suspicious activity detection

## Technologies Used

* Python 3
* os module
* hashlib
* time module
* random module
* string module

## Installation

Clone the repository:

```bash
git clone https://github.com/Kafkacodes/python-hasher-tools.git
cd python-hasher-tools
```

Run any tool:

```bash
python password_strength.py
python password_generator.py
python meta_data.py
python log_analyzer.py
```
